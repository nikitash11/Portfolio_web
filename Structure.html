1.What is a markup?

A document can be defined in terms of its structure; with elements such as headings, paragraphs, lists, tables, diagrams and so on. markup is simply to define these elements and provide a specific tag to markup each element.

2.What is hypertext and hypermedia?

Hypertext is text displayed on a computer display or other electronic devices with references (hyperlinks) to other text that the reader can immediately access, or where text can be revealed progressively at multiple levels of detail (also called StretchText).
Hypermedia is an extension to hypertext providing multimedia facilities, such as those handling sound and video.

3.What is a doctype and why is it important to include it in every HTML document?

Doctype is an instruction that tells Web browsers and other user agents which version of HTML you have used so that they know how to render your pages. It is important to include in every HTML document because the browsers will attempt to render documents without a doctype with their so called "quirck mode" with possible unexpected results.

6.What is a self closing HTML element? Give an example?

The elements which have all the data they need in attribute values without an end tag are called self closing HTML element. For example, the <link tag is self-closed with /> instead of <link>.

7.What is XML and why is it important to web developers?

XML is a a metalanguage which allows users to define their own customized markup languages, especially in order to display documents on the Internet.It is important because it is essential for well structured documents that are easier to develop and maintain.

8.Describe 4 XML rules?

Some XML rules are:

1.Elements must have a start and end tag.
2.Elements must be properly nested.
3.Element names are case-sensitive.
4.Attribute values must be enclosed in double quotation marks.
